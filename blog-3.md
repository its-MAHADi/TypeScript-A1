# How do Generics allow you to build reusable components and functions that stay strictly typed regardless of the data structures passed in?

Generics reusable এবং flexible code লিখতে সাহায্য করে। একই function বা component বিভিন্ন data type এর সাথে কাজ করতে পারে, কিন্তু type safety বজায় থাকে। এতে duplicate code কমে এবং large-scale application maintain করা সহজ হয়।
